## Технические риски (детально)

### R01 — Топология контуров

**Симптом:** буква «О» превращается в сплошной диск или два outer.  
**Причина:** overlapping paths, wrong fill-rule, unclosed beziers.  
**Митигация:**
1. Нормализация winding и `even-odd` vs `nonzero`.
2. Graph-based nesting test (point-in-polygon sampling).
3. UI: «назначить как отверстие» для выделенного контура.

### R02 — Medial axis на сложных формах

**Симптом:** лишние ветки, пропуск тонких перемычек.  
**Митигация:**
- Для ширины < `w_min`: не MAT, а **1D размещение по оси** тонкого штриха.
- Объединение близких веток скелета (pruning длиной < pitch/2).

### R03 — Модель света

**Симптом:** heatmap «красивая», реальность — пятна.  
**Митигация:**
- Параметры `sigma`, `n` (cos power) подбираются по типу рассеивателя.
- Режим «калибровка»: технолог вводит измеренную равномерность → fit коэффициентов.

### R07 — Углы

**Симптом:** тёмные треугольники внутри V, W, K.  
**Митигация:**
- Детектор углов < 90° на offset-границе.
- Дополнительные кандидаты на биссектрисе угла на расстоянии `d = f(angle, depth)`.

| Риск | Митигация |
|------|-----------|
| Один разработчик на core | Документация + CLI tests; code review внешний |
| Затягивание Фазы 3 | Timebox: сначала MAT+offset без оптимизатора |
| Нет обратной связи от цеха | Еженедельные сессии с технологом, 5 макетов/нед |

## План отката (contingency)

1. **Placement не достигает качества:** усилить SemiAuto + библиотека пресетов «для шрифта X».
2. **DXF блокирует сроки:** v1 только SVG + ручной экспорт из Corel.
3. **MAT нестабилен:** fallback на **pure distance-field placement** (всё ещё не «тупая сетка»).
